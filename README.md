# urlChecker

Simple shellscript that "checks" if site exists

## Installation

Just linux shell

## Usage

```python
./urlchecker.sh <file.txt>

./urlchecker.sh urls.txt 
#  Analizando:example.com
#  HTTP/1.1 200 OK
#  Analizando:autodiscover.icbc.com.ar
#  Analizando:bcm.icbc.com.ar
#  ..... 
```
